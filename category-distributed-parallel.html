Parallel News Aggregator

Multithreading • JSONs • Deterministic Output

A multithreaded Java program that processes a large number of news articles from JSON files in an efficient way. Firstly, it removes duplicates(by uuid/title) and then: groups articles by category and language, generates a global chronological list and outputs keyword infomation for english articles.

Distributed Chord Lookup

MPI • Distributed Routing • Finger Table Algorithm

A small distributed lookup program inspired by CHORD. Each MPI process represents a node in the ring, builds its finger table locally and routes lookup requests through message passing until the needed node is found.
