Overview

In this project, I built a suffix tree for a given string (with a '$' terminator), inserting all suffixes one by one. The tree supports substring queries by traversing character by character from the root. In the final part, I implemented a radix compression that merges chains of single-child nodes into one edge labeled with a substring.

C Trees Strings Radix Compression

What I implemented

  • Generated all suffixes of the input string.
  • Inserted suffixes into a tree structure with dynamic children arrays.
  • Supported substring existence queries by traversing edges.
  • Compressed single-child chains into radix edges.

Implementation notes

  • Each node stores its level, character, number of children and children vector.
  • Memory is allocated dynamically for nodes and child arrays.
  • Compression reduces memory usage and traversal depth.
  • Queries run by descending the tree character by character.

Representative code: substring search

The following function checks whether a substring exists in the tree by descending from the root and matching characters on edges. 

int cauta(const char *p, Tree root) {
    const char *q = p;
    Tree cur = root;

    while (*q && cur) {
        int gasit = 0;
        for (int i = 0; i < cur->nr_copii; i++) {
            Tree nxt = cur->children[i];
            if (nxt->caracter == *q) {
                cur = nxt;
                q++;
                gasit = 1;
                break;
            }
        }
        if (!gasit) return 0;
    }
    return 1;
}
Short explanation

The search starts from the root and follows matching edges. If at some point no child matches the next character, then the substring does not exist.
