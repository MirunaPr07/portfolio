Java Airport Scheduler

Runway allocation + urgent priority queues + occupancy rules based on time.

Overview

This project simulates airport runway management. Flights are queued for takeoff or landing, with a separate urgent queue that always has priority. Each runway has rules that determine if it is available, based on the last maneuver and elapsed time.

I kept the logic modular: aircraft types, runway behavior, command parsing and validations are separated, and invalid operations are handled through exceptions.

Java OOP & Inheritance Generics Queues & Scheduling

Key features

  • Runways are categorized by activity: TAKEOFF or LANDING.
  • There are two queues per runway: urgent and normal.
  • Sorting by time/priority (via Comparable and collections).
  • Time of occupancy: runways stay occupied after a maneuver (based on elapsed minutes).
  • Commands for allocation, urgent handling, runway status and flight details.

Implementation notes

  • Used LocalTime, DateTimeFormatter, and Duration for time logic.
  • Custom exceptions for wrong runway selection and for unavailable runway.
  • Runway logic is centralized, while input handling stays in the main command flow.
  • Priority logic is consistent: urgent queue first, then normal queue.

Representative code: the function that handles permission for maneuver

The logic below checks if a runway is still occupied based on the last timestamp. If it is available, it promotes the first eligible aircraft(urgent first), updates status and stores the real timestamp of the maneuver. 

public static void manevra (List<Runway<WideBodyAirplane>> wide_runways, List<Runway<NarrowBodyAirplane>> narrow_runways, String ID_caut, LocalTime timestamp) throws UnavailableRunwayException {
    // nu stiu tipul pistei, tb sa caut in ambele
    DateTimeFormatter formatter = DateTimeFormatter.ofPattern("HH:mm:ss");
    for (Runway<WideBodyAirplane> runway : wide_runways)
        if (runway.getID().equals(ID_caut)) {
            Duration dif = Duration.between(runway.getUltim(), timestamp);
            if ((runway.getAvioane().get(0).getStatus() == Status.WAITING_FOR_TAKEOFF || runway.getAvioane().get(0).getStatus() == Status.DEPARTED) && dif.toMinutes() <= 5)
                throw new UnavailableRunwayException(timestamp.format(formatter) + " | The chosen runway for maneuver is currently occupied");
            else if ((runway.getAvioane().get(0).getStatus() == Status.WAITING_FOR_LANDING || runway.getAvioane().get(0).getStatus() == Status.LANDED) && dif.toMinutes() <= 10)
                throw new UnavailableRunwayException(timestamp.format(formatter) + " | The chosen runway for maneuver is currently occupied");
            for (WideBodyAirplane airplane : runway.getUrgente()) {
                if (airplane.getStatus() == Status.WAITING_FOR_TAKEOFF) {
                    airplane.setStatus(Status.DEPARTED);
                    airplane.setTimp_real(timestamp);
                    runway.setUltim(timestamp);
                    return;
                }
                else if (airplane.getStatus() == Status.WAITING_FOR_LANDING) {
                    airplane.setStatus(Status.LANDED);
                    airplane.setTimp_real(timestamp);
                    runway.setUltim(timestamp);
                    return;
                }
            }
            for (WideBodyAirplane airplane : runway.getAvioane()) {
                if (airplane.getStatus() == Status.WAITING_FOR_TAKEOFF) {
                    airplane.setStatus(Status.DEPARTED);
                    airplane.setTimp_real(timestamp);
                    runway.setUltim(timestamp);
                    return;
                }
                else if (airplane.getStatus() == Status.WAITING_FOR_LANDING) {
                    airplane.setStatus(Status.LANDED);
                    airplane.setTimp_real(timestamp);
                    runway.setUltim(timestamp);
                    return;
                }
            }
        }
}
Short explanation

I search the runway by ID(wide first, then narrow), check occupancy using the last timestamp and a time window, then pick the first eligible airplane (urgent queue first). After the maneuver, status and timestamps are updated.
